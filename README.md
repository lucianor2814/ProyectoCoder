ACTORES, PELICULAS y SERIES conducen, en cada caso, a un formulario donde podremos cargar datos. Si la carga es exitosa se indicará mediante un mensaje. Dichos datos pueden corroborarse mediante el admin.
BUSCAR SERIES POR CANTIDAD DE TEMPORADAS conduce a un formulario donde podremos buscar datos. Si se busca un dato existente en la base de datos, o ningún dato, esto se verá reflejado en un mensaje diferente en cada caso.
